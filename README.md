# Library Management System

A simple Python-based Library Management System developed to manage books, members, and borrowing/return operations using file/CSV handling.

## Features
- Add and view books
- Add and manage members
- Issue books to members
- Return borrowed books
- Store records using CSV/files
- Validate basic user input and record operations

## Technologies Used
- Python
- CSV / File Handling

## Project Purpose
This project was built to practice core Python programming concepts such as functions, control flow, collections, file handling, and validation logic through a real-world management system.

## How to Run
1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code, PyCharm, or terminal.
4. Run the program:

```bash
python main.py
```

## Usage
Use the menu-driven interface to:
- Add book records
- Add member records
- Borrow books
- Return books
- View stored records

## Sample Files
The project uses CSV/files to store data such as books, members, and issued book records.
